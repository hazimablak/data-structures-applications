## 🇹🇷 Türkçe (TR)

### 📌 Proje Hakkında

Bu depo, **C programlama dili** kullanılarak geliştirilmiş temel veri yapıları algoritmalarını ve bu yapıların mantığını pekiştiren uygulamaları içermektedir. Proje, dizilerden ağaç yapılarına kadar geniş bir yelpazede akademik ve pratik örnekler sunar.

### 🗂️ İçindekiler

Klasör içindeki dosyaların işlediği konular:

- **1-diziler.c**: Temel dizi işlemleri ve algoritmaları.
- **2-teklilistler.c**: Tek yönlü bağlı liste (Singly Linked List) yapısı.
- **3-ciftyonlulistler.c**: Çift yönlü bağlı liste (Doubly Linked List) yapısı.
- **4-stackdiziler.c**: Yığın yapısının dizi (Array) ile gerçekleştirimi.
- **5-stacklistler.c**: Yığın yapısının bağlı liste (List) ile gerçekleştirimi.
- **6-queuediziler.c**: Kuyruk yapısının dizi ile gerçekleştirimi.
- **7-queuelistler.c**: Kuyruk yapısının bağlı liste ile gerçekleştirimi.
- **8-trees.c**: İkilik ağaçlar ve veri modelleme.
- **9-stringler.c**: C dilinde karakter dizisi (String) manipülasyonları.
- **10-bankamatik.c**: Veri yapıları kullanılarak hazırlanan ATM simülasyonu.
- **11-giseislemleri.c**: Kuyruk ve liste yapıları ile gişe otomasyon örneği.

### 🚀 Kurulum ve Çalıştırma

Dosyaları derlemek için herhangi bir C derleyicisi (GCC, Clang vb.) kullanabilirsiniz:

```bash
# Örnek derleme
gcc 1-diziler.c -o diziler
./diziler
```

## 🇬🇧 English (EN)

### 📌 About the Project

This repository contains fundamental Data Structures algorithms and practical applications implemented using the C programming language. The project covers a wide range of topics from basic arrays to complex tree structures.

### 🗂️ Table of Contents

Topics covered in the repository:

- **1-diziler.c**: Basic array operations and algorithms.
- **2-teklilistler.c**: Singly Linked List implementation.
- **3-ciftyonlulistler.c**: Doubly Linked List implementation.
- **4-stackdiziler.c**: Stack implementation using Arrays.
- **5-stacklistler.c**: Stack implementation using Linked Lists.
- **6-queuediziler.c**: Queue implementation using Arrays.
- **7-queuelistler.c**: Queue implementation using Linked Lists.
- **8-trees.c**: Binary trees and data modeling.
- **9-stringler.c**: String manipulation in C.
- **10-bankamatik.c**: ATM simulation using data structures.
- **11-giseislemleri.c**: Teller automation example using queues and lists.

### 🚀 Installation & Setup

You can use any standard C compiler (GCC, Clang, etc.) to compile the files:

```bash
# Example compilation
gcc 1-diziler.c -o arrays
./arrays
```
